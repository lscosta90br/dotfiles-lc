# Installation
### Download and change folder
```
cd ~
git clone https://github.com/lscosta90br/dotfiles-lc.git

cd dotfiles-lc
```
### Run install
```
./install.sh
```

# Uninstallation

### Change to dotfiles folder
```
cd ~/dotfiles-lc
```

### Run uninstaller
```
./uninstall.sh
```
