# Installation
### Download and change folder
```
cd ~
git clone https://github.com/lscosta90br/dotfiles.git

ou 

git clone ssh://git@github.com/lscosta90br/dotfiles.git

cd dotfiles
```
### Run install
```
./install.sh
```

# Uninstallation

### Change to dotfiles folder
```
cd ~/dotfiles
```

### Run uninstaller
```
./uninstall.sh
```
